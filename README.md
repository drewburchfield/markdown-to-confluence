## What it does

Converts standard Markdown to Confluence Storage Format (XHTML-based XML). Handles code blocks with syntax highlighting, tables, lists, links, and images. Includes a bundled Python conversion script.

## Install

```
claude plugins install markdown-to-confluence@not-my-job
```
