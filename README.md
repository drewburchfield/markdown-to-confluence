# markdown-to-confluence

Convert Markdown documents to Confluence Storage Format (XHTML-based XML).

A [Claude Code](https://docs.anthropic.com/en/docs/claude-code) plugin from the [not-my-job](https://github.com/drewburchfield/not-my-job) marketplace.

## Features

- Standard markdown syntax
- Code blocks with syntax highlighting
- Tables, lists, links, images
- Bundled Python conversion script

## Install

```
claude plugins install markdown-to-confluence@not-my-job
```
